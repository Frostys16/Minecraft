Крафты

основные рецепты которые нужно знать в майнкрафте

Инструменты

Верстак

4 доски - нужен чтобы крафтить всё остальное

Деревянная кирка

3 доски + 2 палки - для добычи камня

Каменная кирка

3 булыжника + 2 палки - для добычи железа и угля

Железная кирка

3 железных слитка + 2 палки - для добычи алмазов

Алмазная кирка

3 алмаза + 2 палки - для добычи обсидиана

Меч

2 любых материала + 1 палка - для атаки мобов

Броня

Шлем

5 материалов - защищает голову

Нагрудник

8 материалов - защищает тело

Поножи

7 материалов - защищает ноги

Ботинки

4 материала - защищает ноги

Механика

Печь

8 булыжника - для плавки руды и приготовления еды

Сундук

8 досок - для хранения вещей

Кровать

3 доски + 3 шерсти - устанавливает точку возрождения

Факел

1 палка + 1 уголь - освещает пещеры

Зажигалка

1 железный слиток + 1 кремень - зажигает портал в Незер

Ведро

3 железных слитка - носить воду лаву и молоко

совет: всегда носи с собой верстак чтобы крафтить в любом месте
